# FSC SDK

## Description
A simple JavaScript library which provides PWA's the ability to retrieve OAuth
tokens and their related information from PACE native apps.

## Installation
### As NPM package
```bash
yarn add fsc-sdk
```

## Build 
```bash
yarn build fsc-sdk
```

## Type-checking the repo

```sh
yarn run type-check
```

Run in `--watch` mode:

```sh
yarn run type-check:watch
```
